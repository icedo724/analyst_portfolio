배경 및 문제 정의

(주)메디원은 B2B 의료 소프트웨어 공급사로, 수백 개의 병원·의원 거래처를 보유하고 있다. 영업팀은 오랜 기간 담당자 경험에 기반해 거래처 등급(A/B/C)을 부여해왔다.

두 가지 문제가 있었다.

  1. 재현 불가능성 — 같은 거래처를 두 영업 인력이 평가하면 등급이 달라진다.
  2. 신규 인력 의존성 — 경험이 없는 신입이 우선순위를 독립적으로 판단할 기준이 없어, 배정 후 일정 기간 동안 시니어의 안내에 의존해야 했다.

“등급 기준이 사람마다 다르면 CRM 데이터 자체를 신뢰할 수 없다.”

데이터

사내 CRM DB와 매출 데이터를 통합해 분석했다.

출처 포함 항목
CRM DB 거래처 기본 정보, 담당자, 현재 등급
매출 데이터 월별 매출, 결제 이력, 미수금 현황
활동 로그 영업 방문·연락 횟수, 계약 갱신 이력

분석에 사용한 핵심 변수:

  • 재무: 최근 12개월 평균 매출, 결제 지연율, 미수금 비율
  • 관계: 계약 갱신 횟수, 담당자 변경 횟수
  • 활동: 인바운드 문의 빈도, 방문 대비 수주율

분석 방법

1단계 — 데이터 정제 및 변수 구성

CRM과 매출 DB를 거래처 ID 기준으로 조인하고, 결측치를 도메인 맥락에 맞게 처리했다. 예를 들어 미수금 기록이 없는 거래처는 연체 0으로 대체했다.

df['overdue_ratio'] = df['overdue_amount'] / df['total_revenue']
df['overdue_ratio'] = df['overdue_ratio'].fillna(0)

2단계 — 카이제곱 검정으로 등급 판별 변수 선정

현재 등급(A/B/C)과 각 변수 간 독립성을 카이제곱 검정으로 검증했다. 유의수준 α = 0.05 기준:

변수 χ² 통계량 p-value 채택
연간 매출 구간 24.1 < 0.001 ✅
결제 지연율 구간 18.4 0.001 ✅
계약 갱신 횟수 9.2 0.027 ✅
담당자 변경 횟수 3.1 0.212 ❌

담당자 변경 횟수는 “중요할 것”으로 여겨졌지만 통계적으로 등급과 무관함이 확인됐다. 이 결과가 팀 내에서 가장 큰 설득력을 발휘했다.

3단계 — 정량 기준 수립 및 등급 재산정

유의한 변수 3개를 기반으로 점수 체계를 구성하고, 전체 거래처에 재적용했다.

def assign_grade(row):
    score = 0
    if row['annual_revenue'] >= 5000:   score += 3
    elif row['annual_revenue'] >= 2000: score += 2
    else:                               score += 1

    if row['overdue_ratio'] < 0.05:     score += 2
    elif row['overdue_ratio'] < 0.15:   score += 1

    if row['renewal_count'] >= 3:       score += 2
    elif row['renewal_count'] >= 1:     score += 1

    if score >= 6: return 'A'
    elif score >= 4: return 'B'
    else:           return 'C'

핵심 결과

  • 전체 거래처 중 약 30% 의 등급이 기존과 달라졌다.
  • 기존 C등급 → 재산정 A·B등급: 매출 실적은 양호하지만 관계가 오래된 이유로 감으로 저평가된 거래처가 다수 존재했다.
  • 기존 A등급 → 재산정 C등급: 연차가 길어 관성적으로 A를 유지했으나 재무지표가 부진한 거래처가 식별됐다.
  • 신규 인력이 우선순위 판단에 소요하는 시간이 기존 대비 크게 단축됐다(팀장 피드백).

의사결정 연결

분석 결과는 두 가지 실행 변화로 이어졌다.

  1. CRM 등급 필드 업데이트 — 재산정 기준을 CRM 시스템에 반영하여, 이후 신규 거래처도 동일 기준으로 자동 산정되도록 했다.
  2. 영업 주간 미팅 어젠다 변경 — 기존 “이번 주 방문처” 중심에서 “A등급 미방문 거래처 우선 배치” 방식으로 전환됐다.

회고

잘 된 점: 통계 검정으로 “당연히 중요할 것”이라 여겨지던 변수를 탈락시킨 것이 오히려 내부 설득력을 높였다. 데이터로 직관을 반증하는 경험이었다.

한계: 매출 기반 지표만으로는 “성장 가능성 있는 신규 거래처”를 높게 평가하기 어렵다. 장기적으로는 거래처 생애 단계(신규/성장/성숙/이탈)를 추가 변수로 고려해야 한다.
