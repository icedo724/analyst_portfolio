경기도 의왕시

icedo724@gmail.com

데이터 기반 의사결정을 지향하는 분석가입니다.

직관에 의존하던 영역을 정량 기준으로 전환하고, 의사결정권자의 구체적 행동으로 이어지는 결론을 만듭니다. 주요 도구는 Python · SQL · 통계 검정이며, 실무 프로젝트와 다양한 도메인 분석 경험을 보유하고 있습니다.

분석 방향

  • 문제 정의가 출발점입니다. 분석 도구를 적용하기 전에 “어떤 결정에 이 분석이 쓰이는가”를 먼저 정의합니다.
  • 숫자만으로 설명되지 않는 부분이 있습니다. 도메인 맥락과 데이터를 함께 사용해 현상의 원인을 짚어냅니다.
  • 인사이트가 아닌 의사결정으로 끝납니다. 분석의 목적은 보고서가 아니라 다음 행동입니다.

분석 영역

거래처 등급 재조정(B2B), 채용 시장 트렌드, 게임 도메인 (인게임 경제·유저 행동), 시계열·세그먼테이션·통계 검정 등. Projects 페이지에서 상세 확인 가능합니다.
